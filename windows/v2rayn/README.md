# windows/v2rayn/

Extraction target for the v2rayN 7.18.0 bundle downloaded by `windows/setup.ps1`.

## What lands here

| File / pattern | Source |
|---|---|
| `xray.exe`, `v2rayN.exe`, `*.dll` | Extracted from the v2rayN release ZIP by `windows/setup.ps1` |
| `geoip.dat`, `geosite.dat` | Placed by `windows/update_geodata.ps1` (downloaded from GitHub on each run) |
| `config.json` | Generated by `windows/setup.ps1` from repo root `config.json` (three mechanical transforms applied) |

## Git tracking

All runtime files in this directory are gitignored via the wildcard-with-negation
pattern in `.gitignore`:

```
windows/v2rayn/*
!windows/v2rayn/.gitkeep
!windows/v2rayn/README.md
```

Only `.gitkeep` and this `README.md` are committed.
Do not commit binaries — they are ~100 MB and are downloaded on demand by the scripts.
