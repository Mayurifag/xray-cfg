# windows/v2rayn/

Extraction target for the v2rayN bundle downloaded by `windows/setup.ps1`.

## What lands here

| File / pattern                    | Source                                                        |
| --------------------------------- | ------------------------------------------------------------- |
| `xray.exe`, `v2rayN.exe`, `*.dll` | Extracted from the v2rayN release ZIP by `windows/setup.ps1`  |
| `geoip.dat`, `geosite.dat`        | Placed by `windows/update_geodata.ps1` (update every day)     |
| `config.json`                     | Generated by `windows/setup.ps1` from repo root `config.json` |
